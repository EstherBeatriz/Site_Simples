O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação usada para criar e estruturar conteúdo em uma página da web.

O que é CSS?

CSS (Cascading Style Sheets) é usado para estilizar o conteúdo HTML, definindo cores, fontes, espaçamento e layout.
